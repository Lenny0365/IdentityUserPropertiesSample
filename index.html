Identityuserpropertiessample

An MVC 5 project that demonstrates a ASP.NET Identity custom user and use of the entity user store.

Welcome an ASP.NET Identity User Properties Sample

This project is based on the MVC 5 (RTM) template, then extended to show how to capture and store user properties by extending the default User entity, interacting with the external and application cookies, as well as working with the Entity Framework databases used by ASP.NET Identity.

Features

  • Customer User entity.
  • Prompt for additional user details at registration.
  • Allow users to update additional details on account manage page.
  • Requests and captures external login provider user properties.
  • View external login provider user properties on account manage page.
  • All site specific values are appSettings in web.config. (App settings are set in the Azure website config.)

Other polish

  • Custom error pages on Release.
  • Modified Bootstrap theme using Bootswatch.
  • Guest list showing registered users, join date, and connected logins.
  • Google Analytics
